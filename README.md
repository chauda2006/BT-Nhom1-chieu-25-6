# ☕ Anime Coffee Shop Website

Dự án website giới thiệu quán cà phê phong cách Anime.

## 👥 Thành viên & Phân công

* **Châu Tiến Đạt (Nhóm trưởng)**: Tổng hợp & chỉnh sửa mã nguồn.( cố gắng hết sức, lần đầu làm nhóm kiểu này:[)
* **Lý Nguyên Khang**: Trang Chủ (`index.html`).
* **Nguyễn Hoàng Gia**: Trang Dịch vụ (`dichvu.html`).
* **Lê Vũ Lực**: Trang Giới thiệu (`Gioithieu.html`).
* **Võ Khải Toàn**: Trang Liên hệ (`Lienhe.html`).

## 📂 Cấu trúc thư mục

```text
├── assets/
│   ├── icon/
│   ├── image/
│   ├── images/
│   └── mp4/
├── css/
├── js/
├── dichvu.html
├── Gioithieu.html
├── index.html
├── Lienhe.html
└── README.md
```

## 🚀 Cách chạy dự án

1. Mở thư mục dự án bằng phần mềm VS Code.
2. Click đúp chuột vào file `index.html` để xem giao diện trên trình duyệt.
